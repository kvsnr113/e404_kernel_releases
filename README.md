# E404 Kernel

# Supported Device
- Munch  (POCO F4 / REDMI K40S)
- Alioth (POCO F3 / REDMI K30S)
- Apollo (MI10T / MI10T PRO)
- LMI (POCO F2 / REDMI K30 ZOOM)

# Supported Rom
- AOSP/CLO
- MIUI/HyperOS
- Oplus (OOS, COS, MYUI, etc)

# NOTES
- A16 BPF ROMs is not supported !
- Use SUSFS at Your Own Risk !
